# 최소 신장거리
# kruskal 알고리즘
## 특징
- 그리디 알고리즘 활용, 간선을 가중치 순으로 정렬, Disjoint를 활용하여 사이클 방지

## 장점
- 희소그래프에서 효율적, 간선중심 알고리즘, 음의 가중치도 처리 가능

## 단점
- 간선 정렬이 필요, 동적그래프에 사용 어려움

## 시간복잡도
- O(E log E) or O(E log V)

# prim 알고리즘
## 특징
- 그리디 알고리즘, 각 단계에서 가중치가 작은 간선을  선택

## 장점
- 밀집 그래프에서 효율적, 정점 중심의 알고리즘

## 단점
- 희소 그래프에서 비효율적

## 시간 복잡도
- O((V+E) log V)
